1. **소개**

2. **part1-2** : git add, commit으로 파일 기록해놓을 수 있음

9. **part1-9** : Github 사용법 3. 브랜치로 협업하기 (pull request)
10. **part1-10** : git flow / trunk-based 브랜치 전략
11. **part1-11** : git stash로 코드 잠깐 보관하기
